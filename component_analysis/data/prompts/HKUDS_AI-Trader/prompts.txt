DYNAMIC PLACEHOLDERS SUMMARY
=============================

{date}
  - Current date and time provided to the agent

{positions}
  - Current holdings: stock/crypto symbols with quantities and available CASH balance
  - Populated by get_today_init_position(today_date, signature)

{yesterday_close_price}
  - Previous time period closing prices for valuation
  - US/Crypto: Populated by get_yesterday_open_and_close_price()
  - A-Shares: Formatted with Chinese stock names via format_price_dict_with_names()

{today_buy_price}
  - Current buying prices for all assets
  - US/Crypto: Populated by get_open_prices()
  - A-Shares: Populated by get_open_prices() and formatted with Chinese names

{current_profit}
  - Previous time period profit calculation
  - A-Shares only: Populated by get_yesterday_profit()
  - Shows daily profit for day-trading or hourly profit for hour-level trading

{STOP_SIGNAL}
  - Output signal: "<FINISH_SIGNAL>"
  - Agent outputs this when task is complete
